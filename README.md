## 特殊記号

- 撥音 **ん**：文脈により \[n], \[ŋ], \[ɴ]
- 促音 **っ**：直後の無声子音を延長 \[Q]
- 長音：母音の延長 \[aː], \[oː] など
- 外来音補足：小書き母音（ァ, ィ, ゥ, ェ, ォ）、ヴ系列 \[v]、ティ\[ti]、ディ\[di] など
